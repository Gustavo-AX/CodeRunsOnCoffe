LeetCode

LeetCode é um site que reúne problemas de programação de diversos níveis de dificuldade e permite que o usuário envie sua própria solução, na linguagem de programação de sua escolha. O sistema do LeetCode então avalia automaticamente se o código resolve corretamente o problema proposto. Além disso, é possível visualizar como outros usuários solucionaram o mesmo problema e comparar o desempenho da sua solução em termos de tempo de execução e uso de memória. Por isso, o LeetCode é uma excelente ferramenta para quem deseja estudar para entrevistas de emprego na área de tecnologia, se preparar para o mercado de trabalho ou simplesmente manter suas habilidades de programação afiadas.
